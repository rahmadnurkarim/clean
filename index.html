Auto-Scroll & Scan Console Script

Salin skrip di bawah ini untuk mengambil daftar Followers dan Following asli secara otomatis

Skrip ini dirancang khusus untuk mendeteksi jendela modal daftar pengikut/diikuti di Instagram Web Desktop. Skrip akan menggulir (*scroll*) otomatis ke bawah sampai seluruh akun termuat, kemudian mengekstrak data nama pengguna (*username*) mereka ke dalam format JSON yang siap dimasukkan ke dasbor ini.

Jeda tunggu saat memuat akun berikutnya (Rekomendasi: 1000 - 1500 ms)
Membatasi pencarian jika daftar akun Anda sangat besar

Langkah Detail Penggunaan di Instagram Desktop:

  1. Buka profil Instagram Anda (https://instagram.com/nama_anda) di browser komputer (Chrome / Edge).
  2. Klik tulisan "pengikut" (*followers*) atau "diikuti" (*following*) pada profil Anda hingga pop-up daftar akun muncul di tengah layar.
  3. Tekan F12 (atau Klik Kanan -> klik Inspect/Periksa), lalu pilih tab Console.
  4. Tempelkan (*Paste*) skrip yang sudah Anda salin di atas ke dalam konsol, lalu tekan Enter.
  5. Skrip akan secara otomatis menggulir (*scrolling*) daftar pop-up tersebut sampai bawah.
  6. Setelah selesai, konsol akan menampilkan data JSON. **Salin baris JSON tersebut** lalu masukkan ke tombol **"Impor Data Riil"** di halaman simulator ini!

Impor Data dari Konsol Instagram

Pastikan format JSON yang Anda tempel valid (berisi daftar followers dan following yang diekstrak oleh skrip).
